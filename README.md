## 🎙️ About

**I call it facts.** is a podcast and media platform exploring **AI, robotics,
systems, and the frontier of technology** through long-form conversations with
the people building it. New episodes drop on Spotify, and the conversation keeps
going across the socials.

This profile is home to the show's website, content, and the tooling behind it —
built with **React 18 · TypeScript · Vite · Tailwind CSS · Supabase**.
